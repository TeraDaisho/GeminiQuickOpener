# Gemini Quick Opener

Chromeのその場から、Google Geminiを特定のプロンプト付きで素早く起動できるChrome拡張機能です。
閲覧中のページのURLや選択したテキストをコンテキストとして自動的にGeminiに入力し、スムーズなAI対話を開始できます。

## 特徴 🚀

- **コンテキスト注入**:
    - **URL**: 現在開いているページのURLをプロンプトに自動挿入します。
    - **選択テキスト**: Webページ上で選択したテキストをプロンプトに自動挿入します。
- **Deep Link 起動**:
    - URLパラメータを使用してGeminiを起動するため、DOM操作による不安定さがなく、高速かつ確実に動作します。
- **Gem管理機能**:
    - よく使うプロンプトや特定のカスタムGemを登録・管理できます。
    - ポップアップ内の設定画面から簡単にGemの追加・削除が可能です。
- **自動変換**:
    - 通常のGem URL（`https://gemini.google.com/app/gems/...`）を登録しても、自動的に起動用URLに変換して扱います。

## インストール方法 📥

この拡張機能はChromeウェブストアには公開されていません。以下の手順で手動インストールしてください（デベロッパーモード）。

1.  このリポジトリをZIPでダウンロードし、解凍するか、`git clone` します。
2.  Google Chromeを開き、アドレスバーに `chrome://extensions/` と入力して開きます。
3.  右上の **「デベロッパーモード」** をONにします。
4.  左上の **「パッケージ化されていない拡張機能を読み込む」** をクリックします。
5.  ダウンロード（クローン）した `GeminiQuickOpener` フォルダを選択します。

## 使い方 💡

### 基本的な使い方
1.  **ポップアップ**: アイコンをクリックして「Open Gem」ボタンを押す。
2.  **右クリックメニュー**: ページ上で右クリック -> **「Gemini Quick Opener」** をポイントすると、登録したGemの一覧が表示されます。任意のGemをクリックして起動できます。
3.  **ショートカットキー**: `Command+Shift+E` (Mac) / `Ctrl+Shift+E` (Windows) を押すと、即座に**リストの一番上のGem**（デフォルト）が起動します。
    - ※ショートカットキーは `chrome://extensions/shortcuts` から変更可能です。

### Gem（ショートカット）の追加・管理
1.  ポップアップ右上の **設定アイコン (⚙️)** をクリックします。
2.  **Add New Gem** フォームに入力します：
    - **Name**: わかりやすい名前（例: デフォルト、要約用Gemなど）
    - **URL**: GeminiのURL（`https://gemini.google.com/app` またはカスタムGemのURL）
3.  **"Add Gem"** をクリックするとリストに追加されます。
4.  不要なGemは **"Delete"** ボタンで削除できます。

## 技術仕様 🛠️

- **Manifest V3**: 最新のChrome拡張機能仕様に準拠しています。
- **Deep Link (URL Parameter)**: `?prompt_text=` パラメータを使用してプロンプトを渡しています。
- **Permissions**:
    - `storage`: Gem設定の保存に使用。
    - `activeTab` & `scripting`: 現在のタブのURLと選択テキストを取得するために使用（DOMへの書き込みは行いません）。

## プライバシーとセキュリティ 🔒

- **データの送信**: この拡張機能は、ユーザーが入力したデータ（Gemの設定やURL）を外部サーバーに送信しません。
    - Gemの設定はChromeの同期ストレージ（`chrome.storage.sync`）に保存され、ユーザーのGoogleアカウントに紐付いて同期されます。
- **権限の使用**:
    - `activeTab`: 拡張機能のアイコンをクリック、またはメニュー/ショートカットを使用した際、**その時点でのアクティブなタブ**のURLとタイトルを取得するために一時的に使用します。
    - `scripting`: 選択中のテキストを取得するために、アクティブなタブでスクリプトを実行するために使用します。
    - `contextMenus`: 右クリックメニューを表示するために使用します。
- **Deep Link**: Geminiへのアクセスはすべて公式のURL（`gemini.google.com`）を経由します。
